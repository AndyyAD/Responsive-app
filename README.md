## ‚öôÔ∏è Usage

To run this app locally, Use the following commands.

Clone the project

```bash
    git clone https://github.com/AndyyAD/Responsive-app.git
```

Go to the project directory

```bash
    cd Responsive-app
```

Install dependencies

```bash
    npm install --global yarn
```

Run project

```bash
    yarn dev
```
